기본 설정

프롬프트 개선 시 참고할 가이드라인을 설정할 수 있습니다. 예: "간결하고 명확하게", "전문적인 톤 유지" 등

모델 설정

프롬프트 개선에 사용할 AI 모델을 선택하세요. 선택한 모델의 API 키가 필요합니다.

API 키 설정

프롬프트 개선을 위해서는 최소 하나의 AI 모델 API 키가 필요합니다.
Gemini API 키 발급받기
OpenAI API 키 발급받기
Claude API 키 발급받기

Slack 연동

Gemini 응답을 Slack 채널에 공유하기 위한 Incoming Webhook URL입니다.

고급 설정

0.0
응답의 창의성을 조절합니다. 낮을수록 일관적이고 예측 가능하며, 높을수록 창의적이고 다양합니다. (0.0 ~ 2.0)
1.0
다양성 제어 파라미터입니다. 낮을수록 더 집중된 응답을 생성합니다. (0.0 ~ 1.0)
생성되는 응답의 최대 길이를 토큰 단위로 제한합니다. (1 ~ 8192)
3
프롬프트 개선 시 참고할 이전 대화 턴 수입니다. 0으로 설정하면 컨텍스트를 사용하지 않습니다. (0 ~ 5)
